## Pré-requisitos

Antes de começar, você precisa de:

- Um par de credenciais (`API Key` + `API Secret`) fornecido pela equipe Keyspot
- Uma ferramenta para fazer requisições HTTP (curl, Postman, ou sua linguagem preferida)

<Callout kind="info">
  Ainda não tem credenciais? Entre em contato com a equipe Keyspot pelo e-mail **parcerias@keyspot.app** para solicitar acesso.
</Callout>

## Sua primeira chamada em 5 minutos

<Steps>
  <Step title="Verifique se a API está online" icon="activity">
    Faça uma chamada ao endpoint de health check (não requer autenticação):

    <CodeGroup tabs="Bash,TypeScript,Python">
      ```bash
      curl https://partner-api.keyspot.app/health
      ```

      ```typescript
      const response = await fetch("https://partner-api.keyspot.app/health");
      const data = await response.json();
      console.log(data);
      // { "status": "ok", "timestamp": "2026-03-13T10:00:00.000Z" }
      ```

      ```python
      import requests

      response = requests.get("https://partner-api.keyspot.app/health")
      print(response.json())
      # {"status": "ok", "timestamp": "2026-03-13T10:00:00.000Z"}
      ```
    </CodeGroup>
  </Step>

  <Step title="Gere um token de acesso" icon="lock">
    Envie suas credenciais via headers `X-API-Key` e `X-API-Secret`:

    <CodeGroup tabs="Bash,TypeScript,Python">
      ```bash
      curl -X POST https://partner-api.keyspot.app/v1/auth/token \
        -H "X-API-Key: SUA_API_KEY" \
        -H "X-API-Secret: SEU_API_SECRET"
      ```

      ```typescript
      const response = await fetch("https://partner-api.keyspot.app/v1/auth/token", {
        method: "POST",
        headers: {
          "X-API-Key": "SUA_API_KEY",
          "X-API-Secret": "SEU_API_SECRET",
        },
      });

      const { data } = await response.json();
      const token = data.token;
      console.log(`Token: ${token}`);
      console.log(`Expira em: ${data.expiresIn}s`);
      ```

      ```python
      import requests

      response = requests.post(
          "https://partner-api.keyspot.app/v1/auth/token",
          headers={
              "X-API-Key": "SUA_API_KEY",
              "X-API-Secret": "SEU_API_SECRET",
          },
      )

      data = response.json()["data"]
      token = data["token"]
      print(f"Token: {token}")
      print(f"Expira em: {data['expiresIn']}s")
      ```
    </CodeGroup>

    A resposta contém o token JWT válido por 1 hora:

    ```json
    {
      "data": {
        "token": "eyJhbGciOiJIUzI1NiIs...",
        "expiresIn": 3600,
        "tokenType": "Bearer"
      }
    }
    ```
  </Step>

  <Step title="Liste imóveis disponíveis" icon="home">
    Use o token no header `Authorization` e especifique os campos desejados:

    <CodeGroup tabs="Bash,TypeScript,Python">
      ```bash
      curl https://partner-api.keyspot.app/v1/properties\?fields\=code,title,status,operationType,pricing\&limit\=5 \
        -H "Authorization: Bearer SEU_TOKEN_JWT"
      ```

      ```typescript
      const response = await fetch(
        "https://partner-api.keyspot.app/v1/properties?fields=code,title,status,operationType,pricing&limit=5",
        {
          headers: {
            Authorization: `Bearer ${token}`,
          },
        }
      );

      const { data, meta } = await response.json();
      console.log(`Total de imóveis: ${meta.total}`);
      data.forEach((p) => console.log(`${p.code} - ${p.title} (${p.status})`));
      ```

      ```python
      import requests

      response = requests.get(
          "https://partner-api.keyspot.app/v1/properties",
          params={
              "fields": "code,title,status,operationType,pricing",
              "limit": 5,
          },
          headers={"Authorization": f"Bearer {token}"},
      )

      result = response.json()
      print(f"Total de imóveis: {result['meta']['total']}")
      for p in result["data"]:
          print(f"{p['code']} - {p['title']} ({p['status']})")
      ```
    </CodeGroup>
  </Step>

  <Step title="Descubra os campos disponíveis" icon="list">
    Use o endpoint `/v1/fields` para ver todos os campos que você pode solicitar:

    <CodeGroup tabs="Bash,TypeScript,Python">
      ```bash
      curl https://partner-api.keyspot.app/v1/fields \
        -H "Authorization: Bearer SEU_TOKEN_JWT"
      ```

      ```typescript
      const response = await fetch("https://partner-api.keyspot.app/v1/fields", {
        headers: { Authorization: `Bearer ${token}` },
      });

      const { data } = await response.json();
      console.log("Campos de properties:", Object.keys(data.properties));
      console.log("Campos de leads:", Object.keys(data.leads));
      ```

      ```python
      response = requests.get(
          "https://partner-api.keyspot.app/v1/fields",
          headers={"Authorization": f"Bearer {token}"},
      )

      data = response.json()["data"]
      print("Campos de properties:", list(data["properties"].keys()))
      print("Campos de leads:", list(data["leads"].keys()))
      ```
    </CodeGroup>
  </Step>

  <Step title="Crie um lead" icon="user-plus">
    Envie os dados de um novo contato interessado:

    <CodeGroup tabs="Bash,TypeScript,Python">
      ```bash
      curl -X POST https://partner-api.keyspot.app/v1/leads \
        -H "Authorization: Bearer SEU_TOKEN_JWT" \
        -H "Content-Type: application/json" \
        -d '{
          "name": "João Silva",
          "email": "joao@example.com",
          "phone": "11999999999",
          "message": "Tenho interesse neste imóvel.",
          "source": "WEBSITE",
          "propertyCode": "PROP-001"
        }'
      ```

      ```typescript
      const response = await fetch("https://partner-api.keyspot.app/v1/leads", {
        method: "POST",
        headers: {
          Authorization: `Bearer ${token}`,
          "Content-Type": "application/json",
        },
        body: JSON.stringify({
          name: "João Silva",
          email: "joao@example.com",
          phone: "11999999999",
          message: "Tenho interesse neste imóvel.",
          source: "WEBSITE",
          propertyCode: "PROP-001",
        }),
      });

      const { data } = await response.json();
      console.log(`Lead criado: ${data.id} (${data.status})`);
      ```

      ```python
      response = requests.post(
          "https://partner-api.keyspot.app/v1/leads",
          headers={
              "Authorization": f"Bearer {token}",
              "Content-Type": "application/json",
          },
          json={
              "name": "João Silva",
              "email": "joao@example.com",
              "phone": "11999999999",
              "message": "Tenho interesse neste imóvel.",
              "source": "WEBSITE",
              "propertyCode": "PROP-001",
          },
      )

      data = response.json()["data"]
      print(f"Lead criado: {data['id']} ({data['status']})")
      ```
    </CodeGroup>
  </Step>
</Steps>

## Erros comuns no início

<ExpandableGroup>
  <Expandable title="401 — MISSING_CREDENTIALS" default-open="false">
    Você esqueceu de enviar os headers `X-API-Key` e/ou `X-API-Secret` no endpoint de token. Verifique se ambos estão presentes na requisição.
  </Expandable>
  <Expandable title="401 — INVALID_TOKEN" default-open="false">
    O token JWT expirou (válido por 1 hora) ou está malformado. Gere um novo token via `POST /v1/auth/token`.
  </Expandable>
  <Expandable title="400 — VALIDATION_ERROR no fields" default-open="false">
    O parâmetro `fields` é obrigatório nos endpoints de listagem. Consulte `GET /v1/fields` para ver os campos válidos.
  </Expandable>
  <Expandable title="429 — RATE_LIMIT_EXCEEDED" default-open="false">
    Você excedeu o limite de requisições por minuto. Aguarde o tempo indicado no header `Retry-After` antes de tentar novamente.
  </Expandable>
</ExpandableGroup>
