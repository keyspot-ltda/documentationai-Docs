## Overview

Easily start managing docs by creating a new project in Keyspot.

## Steps

1. Log in to your Keyspot dashboard
2. Click "New Project" button in the top menu
3. Enter project name and optional description
4. Click "Create"

Your new project appears on the dashboard and is ready for documentation!
