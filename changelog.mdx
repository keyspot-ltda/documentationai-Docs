## Convenções

- **Nova funcionalidade**: recurso totalmente novo adicionado à API
- **Melhoria**: aprimoramento de funcionalidade existente
- **Correção**: bug fix ou ajuste de comportamento

<Update label="2026-03-13" description="v1.0.0" tags={["Lançamento"]}>
  ### Lançamento inicial da Partner API

  **Novas funcionalidades:**

  - Autenticação JWT em duas fases (`POST /v1/auth/token`)

  - Listagem de imóveis com seleção de campos (`GET /v1/properties`)

  - Listagem de leads com seleção de campos (`GET /v1/leads`)

  - Criação de leads com vinculação automática a imóveis (`POST /v1/leads`)

  - Consulta de campos disponíveis por entidade (`GET /v1/fields`)

  - Consulta de valores distintos para filtros (`GET /v1/values/{entity}/{field}`)

  - Health check (`GET /health`)

  **Recursos de infraestrutura:**

  - Rate limiting por cliente com sliding window de 60 segundos

  - Restrição por IP (whitelist configurável por cliente)

  - Filtros de data (`createdAfter`, `createdBefore`, `updatedAfter`, `updatedBefore`)

  - Paginação offset-based com limite máximo de 100 itens por página

  - Respostas de erro padronizadas com códigos programáticos

  **Documentação:**

  - Documentação completa com guias, API Reference e Postman Collection

  - Dicionário de dados com todas as entidades e enumerações.
</Update>
