## Prerequisites

<Callout kind="info">
Before starting, ensure you have:
- A valid email address for account verification
- A list of properties ready for import (CSV or API format)
- Access to your real estate agency's data (leads, listings)
</Callout>

Keyspot requires a modern web browser like Chrome or Firefox. No software installation needed—everything runs in the cloud.

## Create and Verify Your Account

Follow these steps to get your Keyspot account up and running.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.keyspot.com` and click **Sign Up**.

    Fill in your email, agency name, and create a strong password.

    <Image
      src="https://www.keyspot.com.br/_next/image?url=%2Fimages%2Fkeyspot%2Fmock%2Fhome%2F193632.jpg&w=1200&q=75"
      alt="Keyspot signup form"
      width="800"
      height="400"
    />
  </Step>

  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification email from Keyspot.

    Click the verification link to activate your account.
  </Step>

  <Step title="Log In" icon="log-in">
    Return to the dashboard and log in with your credentials.

    You'll land on the main dashboard.
  </Step>
</Steps>

## Set Up Your Dashboard and Import Properties

Once logged in, configure your dashboard and add your first property.

<Steps>
  <Step title="Complete Profile" icon="settings">
    Navigate to **Agency Settings** > **Profile**.

    Add your agency details, logo, and contact information.
  </Step>

  <Step title="Import Properties" icon="upload">
    Go to **Properties** > **Import**.

    Choose your method: CSV upload or API integration.

    <Tabs>
      <Tab title="CSV Upload" icon="file-text">
        Download the sample CSV template and fill it with your data.

````csv
name,price,bedrooms,bathrooms,location
"Apartamento Centro",250000,3,2,"São Paulo, SP"
"Casa Jardim",450000,4,3,"Rio de Janeiro, RJ"
````

        Upload the file and map columns if needed.
      </Tab>

      <Tab title="API" icon="code">
        Use the Properties API to push data programmatically.

        <CodeGroup tabs="JavaScript,Python">
````javascript
const response = await fetch('https://api.example.com/properties', {
  method: 'POST',
  headers: { 'Authorization': `Bearer YOUR_API_KEY`, 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'Apartamento Centro',
    price: 250000,
    bedrooms: 3,
    bathrooms: 2,
    location: 'São Paulo, SP'
  })
});
````
````python
import requests

response = requests.post(
  'https://api.example.com/properties',
  headers={'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json'},
  json={
    'name': 'Apartamento Centro',
    'price': 250000,
    'bedrooms': 3,
    'bathrooms': 2,
    'location': 'São Paulo, SP'
  }
)
````
        </CodeGroup>
      </Tab>
    </Tabs>
  </Step>
</Steps>

<Callout kind="tip">
After import, properties appear in your dashboard. Review and edit any mappings.
</Callout>

## Configure Basic AI Website Features

Enable AI-powered features for your site.

<Steps>
  <Step title="Activate AI Site" icon="zap">
    Go to **Website** > **AI Settings**.

    Toggle **AI Recommendations** to `on`.

    Set preferences like price range filters.
  </Step>

  <Step title="Customize Vitrine" icon="monitor">
    In **Website Builder**, select **Vitrine Inteligente**.

    Add filters for bedrooms, location, and price.

    Preview your live site at `https://yoursite.keyspot.com`.
  </Step>
</Steps>
